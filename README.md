# Adoption Manager
> Jest to aplikacja mobilna służąca do przeglądania ogłoszeń o zwierzętach laboratoryjnych dostępnych do adopcji. Zawiera autoryzację przez Firebase, dynamicznie renderowane widoki.
> Chwilowo opiera się na mockowanych danych

## Wprowadzenie 
 
Aplikacja mobilna do przeglądania ogłoszeń wystawianych przez organizacje zamujące się adopcją zwierząt laboratoryjnych.<br/>
Niezalogowany użytkownik nie można przeglądać listy. Po zalogowaniu się można oglądać oferty, wchodzić w ich szczegóły oraz docelowo adoptować. Nie może
dodawać nowych zwierząt, jest to docelowo możliwe tylko dla organizacji. <br/>
Użytkownik może się logować za pomocą emaila i hasła.<br/>

## Uruchomienie

<b>-npm install</b>
<b>-npm start</b>
